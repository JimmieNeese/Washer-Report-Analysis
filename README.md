# Washer-Report-Analysis

## Objective

The Washer Analysis Report Project was done to enhance my skills on investigating a forensic image

### Skills Learned

- Proficiency in analyzing a forensic image.
- Proficiency in using Autopsy and FTK Imager
- Proficiency in creating a professional report.
- Development of critical thinking and problem-solving skills in cybersecurity.

### Tools Used

- FTK Imager - Used to create a Bit-by-Bit copy of the hard drive
- Autopsy - Used to investigate the image
- Windows Registry Explorer - Used to investigate the Hives found on the image
- PrefetchWindows Event Viewer - Used to investigate the event log found on the image
